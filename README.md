# eBumenyi Dev Work

Monorepo for the eBumenyi Community Health Worker platform.

## Repository

https://github.com/housemajorrwanda/eBumenyi-dev-work

## Projects

| Directory | Description |
|-----------|-------------|
| [`api/`](./api) | Node.js / Express / Prisma backend API |
| [`web/`](./web) | React web admin and eTraining portal |
| [`mobile/`](./mobile) | React Native / Expo mobile app |
| [`meet/`](./meet) | Next.js video conferencing app |

## Getting Started

Clone the repository:

```bash
git clone https://github.com/housemajorrwanda/eBumenyi-dev-work.git
cd eBumenyi-dev-work
```

Then open the README in the project you want to run:

- [API setup](./api/README.md)
- [Web setup](./web/README.md)
- [Mobile setup](./mobile/README.md)
- [Meet setup](./meet/README.md)
