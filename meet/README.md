# eBumenyi Meet

Next.js video conferencing app for RBC meetings and scheduled sessions.

Part of the [eBumenyi-dev-work](https://github.com/housemajorrwanda/eBumenyi-dev-work) monorepo.

## Getting Started

Clone the monorepo and move into the meet app:

```bash
git clone https://github.com/housemajorrwanda/eBumenyi-dev-work.git
cd eBumenyi-dev-work/meet
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```
