# CHW Mobile App

React Native / Expo mobile application for the eBumenyi Community Health Worker platform.

Part of the [eBumenyi-dev-work](https://github.com/housemajorrwanda/eBumenyi-dev-work) monorepo.

## Getting Started

Clone the monorepo and move into the mobile app:

```bash
git clone https://github.com/housemajorrwanda/eBumenyi-dev-work.git
cd eBumenyi-dev-work/mobile
```

Install dependencies:

```bash
npm install
```

Start the Expo development server:

```bash
npx expo start
```
